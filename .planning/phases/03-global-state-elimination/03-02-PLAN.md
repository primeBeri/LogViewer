Key ViewModel properties relevant to new tests:
- IsPaused (bool) — triggers buffer/flush in OnLogEventAsync
- LogHandleFilter (string) — wildcard or regex; ".*" matches all; SetRegexFilterIfValid("invalid((") returns false
- LogLevel (LogLevel) — filters events below minimum severity (unless ExactLogLevelFilter)
- MaxLogSize (int) — collection trimmed to this limit after each add
- LogEvents (ReadOnlyLogCollection) — the observable collection asserted against
- PauseBufferCount (int) — count of events currently buffered while paused

How events reach the VM:
1. Call sink.Write(new LogEventArgs(...)) — TestBaseLoggerSink fires LogReceived synchronously
2. VM's OnLogEventAsync runs synchronously via FakeDispatcher
3. await Task.Delay(50) gives async continuations time to complete

LogEventArgs constructor: new LogEventArgs(LogLevel level, string handle, string message, LogColor color)
LogLevel enum: LogLevel.Trace(0) < Debug(1) < Information(2) < Warning(3) < Error(4) < Critical(5)
