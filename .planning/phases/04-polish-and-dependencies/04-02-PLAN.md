<objective>
Make LogWindow a functional standalone window by embedding LogControl inside its Grid,
and fix LogControl's hardcoded white background so it inherits the host application's
WPF theme.

Purpose: LogWindow currently ships as an empty Grid shell — calling new LogWindow().Show()
shows a blank window. UI-01 requires it to host a working log viewer. LogControl's
Background="White" breaks dark-themed host applications (UI-02).

Output: LogWindow.xaml with <local:LogControl /> inside its Grid; LogControl.xaml with
Background="{DynamicResource {x:Static SystemColors.WindowBrushKey}}" on the root Grid.
No code-behind changes are needed for either file.
</objective>
