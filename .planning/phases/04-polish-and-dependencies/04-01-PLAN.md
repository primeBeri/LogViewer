<objective>
Remove Newtonsoft.Json from the library and test project entirely, replacing JSON
serialization in LogExporter with System.Text.Json. The replacement must produce
indented JSON with LogLevel serialized as a string enum name rather than its integer
value. The existing test suite must pass without modification to test intent.

Purpose: Newtonsoft.Json is a transitive NuGet dependency for every consumer of
RealTimeLogStream. Eliminating it reduces the install footprint and removes a heavy
legacy dependency that adds no value over the inbox System.Text.Json runtime.

Output: LogViewer.csproj with no Newtonsoft.Json reference, LogExporter using
System.Text.Json, LogEventArgs using System.Text.Json.Serialization.JsonIgnore,
LogEventArgsMap with clean imports, and LogExporterTests asserting the new
System.Text.Json output shape.
</objective>
